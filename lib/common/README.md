# common/

Shared types, logging, error codes, FreeRTOS queue wrappers.

| File | Purpose |
|------|---------|
| `types.h` | `DeviceId`, `Timestamp`, result types |
| `errors.h` | `LoRaLinkError` enum |
| `logger.h` | Leveled serial logging (INFO/DEBUG/WARN/ERROR) |
| `queues.h` | Typed queue factory helpers |
